**The Last Honest Meter** (`the_last_honest_meter.md`) — the
first-playable seed work (Kernos letter 016). Original to the
pattern-buffer project (authored for its chapter test), NOT
public-domain — fully owned, no licensing question. The canonical
demonstration world across the mesh: fiction-ingestion eval,
`anchor.world` demo, and Holodeck first playable.

**The firewall (letter 016 guardrails — binding):**

1. The prose here is a copy of PB's `evals/last_honest_meter/story.md`
   (with its `SEED_VERSION` stamp). **The bible (`bible.md`) stays in
   PB's repo as the grading reference and is NEVER copied here and
   NEVER ingested** — the same firewall the chapter test enforces. If
   the bible appears in this directory, something is wrong; delete it
   and check how it got here.
2. First playable ingests this prose **fresh through Holodeck's own
   session-zero pipeline** (ingest → scenario → play, end to end) —
   never by loading PB's stamped `anchor.world`.

**Novel-arc mode (letter 016):** for this world, the arc's
`climax_ready` sufficiency set IS the mystery's solution set — the
`achievable_via` machinery binds directly to the authored evidence
graph. No separate arc authoring; the mystery is the arc.
